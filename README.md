# Новостной портал

Статическая верстка главной страницы новостного портала с адаптивным дизайном и интерактивными элементами

## Функциональность

- **Адаптивный дизайн** 
- **Интерактивные новостные карточки** с плавным раскрытием/скрытием анонсов
- **Блок популярных новостей** с навигацией по клику
- **Hover-эффекты** с анимацией для карточек
- **Семантическая верстка** с использованием HTML5 тегов

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - Grid, Flexbox, анимации, медиа-запросы
- **JavaScript** - работа с DOM, обработка событий

## Особенности реализации

### Семантическая структура
- `<header>` - шапка с навигацией
- `<main>` - основной контент
- `<article>` - новостные карточки
- `<aside>` - блок популярных новостей
- `<footer>` - подвал сайта

### Адаптивность
- CSS Grid для сетки новостей
- Flexbox для основной компоновки
- Медиа-запросы для мобильных устройств

### Интерактивность
- Кнопки "Читать далее/Свернуть" с плавной анимацией
- Взаимодействие с блоком популярных новостей
- Визуальное выделение при наведении

## Запуск сайта через локальный сервер

1. Откройте терминал/PowerShell в папке, где лежат `index.html` и `news.txt`

2. Запустите команду: `python -m http.server 8000 `
